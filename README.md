# mocom11ty

## Getting Started

Install all dependencies using npm:

```
$ nvm use
$ npm install
```

### To Develop

```
$ npm start
```

 And in debug mode:
 
```
$ npm run debugstart
```

You can view the website locally at the given URL:

```
$ [11ty] Server at http://localhost:8081/
```

### To Build

```
npm run build
```
